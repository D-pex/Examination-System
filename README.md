# 📘 Examination System

A scalable and modular **Online Examination System** designed to streamline digital assessments with efficiency, security, and automation.

---

## 🚀 Overview

The **Examination System** is a web-based platform that enables institutions to conduct, manage, and evaluate exams digitally. It reduces manual effort by automating exam creation, evaluation, and result processing.

---

## 🎯 Key Features

- 🔐 **Authentication & Authorization**
  - Secure login system for Admin and Students

- 📝 **Exam Management**
  - Create, update, and delete exams
  - Set time limits and rules

- 📚 **Question Bank**
  - Add, edit, and organize questions

- ⏱️ **Timed Exams**
  - Automatic submission when time expires

- 📊 **Auto Evaluation**
  - Instant result calculation

- 📈 **Performance Tracking**
  - View results and exam history

- 🧑‍💼 **Admin Panel**
  - Manage users, exams, and data

---

## 🏗️ System Architecture

```
Client (Frontend)
      ↓
Controllers / API Layer
      ↓
Business Logic Layer (Services)
      ↓
Database
```

- Follows layered architecture
- Clean separation of concerns
- Scalable and maintainable design

## ⚙️ Installation & Setup

### 1. Clone Repository
```bash
git clone https://github.com/D-pex/Examination-System-.git
cd Examination-System-
```

### 2. Install Dependencies
```bash
npm install
# OR
dotnet restore
```

### 3. Configure Environment
- Set database connection string
- Configure environment variables if required

### 4. Run Application
```bash
npm start
# OR
dotnet run
```

## 🧪 Usage Flow

1. Admin logs in  
2. Creates exams and adds questions  
3. Students log in  
4. Attempt exams within time limit  
5. System evaluates automatically  
6. Results are displayed instantly
